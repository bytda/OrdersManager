Веб-приложение на ASP.NET Core с использованием Entity Framework Core и базы данных SQLiteДля работы с приложением, необходимо скачать файлы проекта и запустить на своем компьютере, для мобильности проекта, подключена база данных SQLite, поэтому подключать дополнительно ничего не нужно. Данное приложение позволяет оформлять заказы доставки а также видеть перечень уже оформленных заказов, чтобы приступить к работе нажмите в верхнем левом углу на кнопку "Заказы". Для удобства ознакомления с функционалом приложения пару заказов оформлены заранее.
